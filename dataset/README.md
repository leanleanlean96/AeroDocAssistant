# Датaсет AeroDoc Assistant

## Описание
Симулированный датасет технической документации авиастроительного предприятия для разработки и тестирования AI-ассистента.

## Структура
aero_dataset/
├── 1_structured/ # Структурированные данные (JSON/XML)
├── 2_unstructured/ # Неструктурированные документы (PDF/DOCX)
├── 3_glossary/ # Словарь терминов и сокращений
└── README.md # Это описание


## Содержание
- **42 документа** различных типов
- **10+ спецификаций** в JSON формате
- **6 стандартов** в XML/JSON форматах
- **5 чертежей** (текстовые PDF)
- **3 руководства** (DOCX)
- **4 отчёта** (PDF/DOCX)
- **3 технологических процесса** (DOCX)
- **Словарь** из 50+ терминов и сокращений

## Особенности датасета
1. **Реалистичные связи** между документами
2. **Преднамеренные противоречия** для тестирования системы:
   - Разные значения моментов затяжки в STD-078 и STD-189
   - Устаревшая спецификация SPEC-FASTENERS-009
3. **Граф связей** в relations.json для визуализации
4. **Метаданные** всех документов в metadata.csv

## Использование
Датасет предназначен для:
1. Разработки RAG-системы семантического поиска
2. Тестирования анализа противоречий
3. Построения графа знаний
4. Демонстрации работы AI-ассистента

## Форматы
- JSON: Спецификации, часть стандартов
- XML: Стандарты, структурированные данные
- PDF: Чертежи, отчёты (текстовое содержание)
- DOCX: Руководства, техпроцессы
- CSV: Метаданные, сокращения
